# 🍽️ Iloilo Reserve Food

**Iloilo Reserve Food** is a backend-driven reservation and pre-ordering system designed to modernize the restaurant dining experience through efficiency, automation, and convenience.

Originally developed as part of a **Capstone Project**, this system is engineered with a **real-world application in mind**—addressing common inefficiencies in food service operations such as long queues, order delays, and seat unavailability.

---

## 🌟 Overview

In many restaurant environments, customers often face:
- Long waiting times at ordering counters  
- Delayed food preparation  
- Difficulty securing available seating  

**Iloilo Reserve Food** redefines this experience by enabling users to:
- 📦 Pre-order meals before arriving  
- 🪑 Reserve seats in advance  
- ⏱️ Eliminate waiting time upon arrival  

By the time customers reach the restaurant, **their orders are already being prepared (or ready), and their seats are secured**, creating a smooth and premium dining experience.

## 🏗️ System Architecture

This project is built using the **MEN Stack**:

- **MongoDB** – Data storage for orders, reservations, and users  
- **Express.js** – RESTful API and business logic  
- **Node.js** – Server runtime environment  

> Designed as a **backend-first system**, it can be integrated with web, mobile, or kiosk-based frontends.
